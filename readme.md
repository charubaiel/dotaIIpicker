#  Запуск ETL процесса по выгрузке данных.  

```
docker build -t dotaetl .
docker run \
    --name dotaetl_container \
    -p 3000:3000 \
    -e STEAM_API_KEY=$STEAM_API_KEY \
    -v $PWD/ETL/dbs:/opt/dagster/app/dbs \
    -d dotaetl
```
Морда дагстера на  localhost:3000
